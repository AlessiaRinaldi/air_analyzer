Air Analyzer

Air Analyzer
Wearable Smart Mask for Breathing Activity Monitoring

Air Analyzer is an embedded system designed to monitor breathing-related activity using a microcontroller-based architecture.

The project focuses on sensor acquisition, firmware development, interrupt-based logic, and documentation of the embedded workflow.
